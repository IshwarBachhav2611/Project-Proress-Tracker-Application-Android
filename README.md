# Project Progress Tracker

## 📌 Overview

Project Progress Tracker is an Android-based application developed to simplify and organize the management of student mini projects. The application replaces traditional manual tracking methods such as registers and spreadsheets with a structured, role-based mobile solution.

The system allows Admins, Teachers, and Students to manage and monitor project activities efficiently through a user-friendly Android interface.

---

## ✨ Features

### 👨‍💼 Admin

* Create and manage classes
* Generate unique class codes
* Manage Teachers and Students

### 👨‍🏫 Teacher

* View student groups
* Manage project-related information

### 👨‍🎓 Student

* Login using class code
* View assigned group details
* Access project information

### 🔐 Authentication

* Secure login validation
* Class code verification

---

## 🛠️ Technologies Used

* **Java** – Core programming language
* **Android Studio** – Development environment
* **XML** – UI Design
* **Shared Preferences** – Local data storage

---

## 📱 System Architecture

The application follows a role-based structure:

```text
Admin → Creates Classes & Manage Users
Teacher → Manages Groups, Assign tasks, check progress
Student → Views Update Project Details and Create groups
```

## 🎯 Objectives

* Improve transparency and coordination
* Reduce manual record-keeping
